# 📚 知识库 (Knowledge Base)

这是豪哥的个人知识管理系统，由 Jarvis 协助管理。

## 目录结构

```
knowledge-base/
├── notes/       # 永久笔记 - 经过整理的知识点
├── daily/       # 每日笔记 - 日常记录、灵感
├── projects/    # 项目笔记 - 按项目组织的文档
├── resources/   # 资源收藏 - 链接、书籍、文章摘要
└── archive/     # 归档 - 不常用但想保留的内容
```

## 使用方式

直接跟 Jarvis 说：
- "帮我记一条笔记：xxx"
- "搜索关于 xxx 的笔记"
- "整理一下我最近的笔记"
- "把这个链接保存到资源里"
- "看看我的知识库概览"

## 笔记格式

每条笔记使用 Markdown 格式，包含 YAML front matter：

```markdown
---
title: 笔记标题
tags: [tag1, tag2]
created: 2026-01-30
updated: 2026-01-30
---

# 标题

内容...
```
