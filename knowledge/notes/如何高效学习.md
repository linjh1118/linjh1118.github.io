# 如何高效学习

## 核心原则

1. **主动回忆** - 不要只是重复阅读，要主动测试自己
2. **间隔重复** - 利用遗忘曲线，定期复习
3. **关联学习** - 把新知识和已有知识建立联系
4. **费曼技巧** - 用简单的话解释给别人听

## 实践方法

- 学完一个概念后，合上书试着复述
- 使用 Anki 等工具做间隔重复
- 画思维导图建立知识关联
- 写博客或笔记来输出
